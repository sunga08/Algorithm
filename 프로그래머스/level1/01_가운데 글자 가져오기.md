## 문제
단어 s의 가운데 글자를 반환하는 함수, solution을 만들어 보세요. 단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.
- s는 길이가 1 이상, 100이하인 스트링입니다.

## 코드
```
#include <string>
#include <vector>

using namespace std;

string solution(string s) {
    string answer = "";
    int len = s.size();
    if(len%2 == 0){   
        answer = s.substr(len/2-1,2);
    }
    else{
        answer = s.at(len/2);
    }
    
    return answer;
}
```
