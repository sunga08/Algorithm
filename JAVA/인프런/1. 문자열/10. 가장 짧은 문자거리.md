## 가장 짧은 문자거리

🐱(🔺)
```java
public void solution(String str, char ch){
    int[] arr = new int[str.length()];

    int p = 1000;
    for(int i=0;i<str.length();i++){
        if(str.charAt(i)!=ch){
            arr[i] = p;
        }
        else{
            p = 0;
        }
        p++;
    }

    for(int i=str.length()-1; i>=0; i--){
        if(str.charAt(i)!=ch && arr[i]>p){
            arr[i] = p;
        }
        else if(str.charAt(i)==ch){
            p = 0;
        }
        p++;
    }

    for(int dd : arr){
        System.out.print(dd+" ");
    }

}
```

- 첫번째 Loop(정방향) : 왼쪽 e와의 거리 계산 (첫번째는 무한대, 문자와 일치하면 0)
- 두번째 Loop(역방향) : 오른쪽 e와의 거리 계산 → 배열에 있는 숫자와 현재 거리 중 작은 숫자 넣기
```java
public void solution(String str, char ch){
    int[] arr = new int[str.length()];

    int p = 1000;
    for(int i=0;i<str.length();i++){
        if(str.charAt(i)==ch){
            p = 0;
            arr[i] = p;
        }
        else{
            p++;
            arr[i] = p;
        }
    }

    p = 1000;
    for(int i=str.length()-1; i>=0; i--){
        if(str.charAt(i)==ch){
            p = 0;
        }
        else {
            p++;
            arr[i] = Math.min(arr[i], p);
        }
    }

    for(int dd : arr){
        System.out.print(dd+" ");
    }

}
```
